# API Error Codes

## AUTH

- `AUTH_DISABLED`: authentication endpoints are disabled
- `AUTH_UNAUTHORIZED`: missing or invalid auth token
- `AUTH_INVALID_CREDENTIALS`: invalid login payload

## PERM

- `PERM_PROJECT_READ`
- `PERM_PROJECT_WRITE`
- `PERM_PROJECT_MANAGE`
- `PERM_ORG_READ`
- `PERM_ORG_WRITE`
- `PERM_ORG_MANAGE`

## CONFLICT

- `PROJECT_STATE_CONFLICT`
- `IDEMPOTENCY_KEY_REUSED`

## VALIDATION

- `VALIDATION_ERROR`
- `PROJECT_CREATE_INVALID`
- `PROJECT_MEMBER_INVALID`
- `PROJECT_COMMIT_INVALID`

## PLATFORM

- `RATE_LIMIT_EXCEEDED`
- `LEGACY_SESSION_DISABLED`
- `LEGACY_SESSION_AUTH_REQUIRED`
- `INTERNAL_ERROR`
